# Inventario Macchine

Questo progetto è stato sviluppato passo dopo passo con l'aiuto di un'Intelligenza Artificiale.

## Idea iniziale
Creare un programma che gestisca un inventario di macchine, memorizzando marca e valore e calcolando il costo totale.

## Sviluppo
Il programma è stato migliorato progressivamente:
- uso di struct
- gestione di più macchine
- calcolo del costo totale
- gestione delle stringhe con spazi
